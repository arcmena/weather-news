### Weather App

This is a weather app! Check the weather at your location!

## Getting started

Clone this repo

```bash
git clone https://github.com/arcmena/weather-news.git
```

Install dependencies

```bash
yarn install

#or

npm install
```

Setup environment variables

```bash
REACT_APP_API_KEY=  # Open Weather Map api key
```

Start the server

```bash
yarn start
```

Run tests

```bash
yarn test
```
